# Installation Guide For Project Elixir on haydn

### IF YOU ARE MOVING FROM ANOTHER ROM, CLEAN FLASH Is COMPULSORY.

- Use Pe or crdroid recovery
- Format data
- Flash latest global, India or cn based firmware
- Reboot to Recovery
- Flash rom zip
- Reboot to System
