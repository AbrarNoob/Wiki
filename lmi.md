# Installation Guide For Project Elixir on POCO F2 Pro (lmi)

### IF YOU ARE MOVING FROM ANOTHER ROM, CLEAN FLASH Is COMPULSORY.

- Format data
- Flash firmware 12.5.3.0.RJKMIXM or 12.5.4.0.RJKCNXM
- Flash rom zip
- Reboot to System
